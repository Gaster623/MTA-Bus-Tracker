# MTA-Bus-Tracker

This application is a bus tracker, using June 2017 data, that can be used to track every
bus in NYC from June 1-30, 2017.

To access the Java file:
MTA-Bus-Tracker/Java-Frontend/src/Main.java

To access the SQL file:
MTA-Bus-Tracker/MySQL-BackEnd/MTA.sql

For the link to a video demo:
MTA-Bus-Tracker/misc/demo-link.txt
